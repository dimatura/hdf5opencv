
hdf5opencv
----------

License: GPLv3

Requirements:

- OpenCV 2.0 or greater (http://opencv.willowgarage.com)
- Boost filesystem library (tested with 1.38) (http://www.boost.org)
- HDF5 library (tested with 1.6.6) (http://www.hdfgroup.org/HDF5)

See ``demo.cc`` for an example.
